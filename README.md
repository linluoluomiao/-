项目简介
本项目为《网络安全与系统设计》课程设计作品。通过 Cisco Packet Tracer 6.2 模拟了一个典型的高校校园网环境，重点实现了包过滤防火墙的区域隔离、服务发布及隐匿防御功能。

网络拓扑架构
方案采用“三向防火墙”架构，将网络逻辑划分为三个安全等级：
Inside (高信任区/办公区)：VLAN 10，存放行政办公核心资产。
DMZ (中立区/服务区)：存放学校官网服务器，对外提供 Web 服务。
Outside (低信任区/互联网)：模拟外部不安全的公网环境。
VLAN 20 (学生教学区)：受限访问区，与办公区实施逻辑隔离。

技术核心与实现
1.包过滤策略 (ACL)
非对称隔离：利用扩展 ACL 区分 ICMP 请求与应答报文，实现“办公区可管教学区，教学区不可探测办公区”的单向访问逻辑。
状态感知模拟：使用 established 关键字在包过滤架构下实现基本的 TCP 状态检测，仅允许外部响应流量回流。
2.地址转换 (NAT/NAPT)
Static NAT：将 DMZ 官网服务器映射至公网 IP 202.100.1.1:80。
Dynamic NAPT：实现内网主机共享公网 IP 访问互联网，同时隐藏内部拓扑结构。

运行环境
软件版本：Cisco Packet Tracer 6.2 (Student Version 或更高)
注意：由于 Packet Tracer 存在版本向下兼容性问题，建议使用 6.2 版本以获得最佳体验。
